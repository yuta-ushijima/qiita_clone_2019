# qiita_clone_2019
This is simple clone app of Qiita

## Getting Started!


### Prerequisites

```
Ruby...2.6.1
Rails...5.2.3
```

### Installing dependence

```
$ bundle install --path vendor/bundle
```

## Running the tests

```
$ bundle exec rspec
```

### And coding style tests
